# Capstone-EdgeSafe
**Edge-Based Firearm Detection and Secure Event Logging System**

## Overview
EdgeSafe is a capstone project that implements an edge-based firearm detection system designed for educational environments. The system uses a deep learning object detection model to detect firearms in images and logs detection events with relevant metadata.

The system runs on an edge device and provides a backend API and web dashboard that allows authorized users to review detection events and associated image artifacts.

## Key Features
- Firearm detection using a YOLO-based object detection model
- Edge-based inference to reduce latency and protect privacy
- Secure event logging with metadata (timestamp, confidence score, image reference)
- Local database storage using SQLite
- Backend API for retrieving detection events
- Dashboard interface for reviewing logged events
